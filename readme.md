## Структура проекта

```

│   ├── data_3                        # 1-31 категория статей
│   │   └── *.txt
│   ├── data_3_1                      # 32-62 категория статей
│   │   └── *.txt
│   └── info                          # Метаданные статей
│        └── *.csv
│
├── data/                             # Собранный корпус научных текстов из .txt
│   └── df.parquet                    # Объединённый датасет
│
├── models/                           # Сохраненные обученные модели
│   ├── cosine_mlp.pt
│   └── pair_mlp.pt
│
├── reports/                          # Результат выполнения
│   └── comparison_report.txt         # Итоги сравнения различных моделей
│
├── nlp_ml_similarity.ipynb           # Рабочий Jupyter Notebook
│
├── requirements.txt                  # Python-зависимости
│
└── tfidf_vectorizer.joblib           # Сохраненный векторизатор текста

```
## Функции проекта
- Очистка, нормализация и предобработка научных текстов  
- Генерация TF-IDF признаков  
- Вычисление косинусного сходства  
- Классификация пары текстов двумя моделями:  
  - **CosineMLP** — по признаку косинусного сходства  
  - **PairMLP** — по двум векторным признакам  
- Сохранение и загрузка обученных моделей
- Возможность сравнения текстов из файлов или с помощью ручного ввода
- Автоматическое формирование отчётов

## Модели

### CosineMLP
Многослойный перцептро, принимающий косинусное сходство между TF-IDF векторами.

### PairMLP
Многослойный перцептрон, обученный на двух признаковых векторах.
